# Phase 3 SEO Optimization - H1 Tags and Content Structure Strategy

## Overview

This document outlines the H1 tag optimization strategy for OXEC Immigration website. H1 tags are critical for SEO as they tell search engines what a page is about. Each page should have exactly ONE H1 tag that includes the primary target keyword.

---

## H1 Tag Optimization Strategy

### Principles

1. **One H1 per page** - Each page should have exactly one H1 tag
2. **Include primary keyword** - H1 should contain the main target keyword for that page
3. **Natural language** - H1 should read naturally, not be keyword-stuffed
4. **Match page intent** - H1 should clearly communicate the page's main topic
5. **Consistent with Title** - H1 should align with the page's Meta Title for consistency

## Content Structure Optimization Guidelines

### H2 Tags (Secondary Headings)

H2 tags should break down the page content into logical sections. Each H2 should:
- Represent a major topic or section
- Include related keywords naturally
- Be descriptive and clear
- Typically 3-5 H2 tags per page

### H3 Tags (Tertiary Headings)

H3 tags should further break down H2 sections. Each H3 should:
- Represent a subsection or detailed point
- Support the parent H2 topic
- Include related keywords where natural
- Be used sparingly (only when needed for clarity)

### Best Practices

1. **Hierarchy** - Maintain proper heading hierarchy (H1 > H2 > H3)
2. **Keyword inclusion** - Include target keywords naturally in headings
3. **Readability** - Keep headings concise and clear
4. **User intent** - Align headings with user search intent
5. **Consistency** - Use consistent heading styles across pages
