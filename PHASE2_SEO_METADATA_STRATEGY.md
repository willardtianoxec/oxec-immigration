# 第二阶段SEO优化 - 元数据优化策略

本文档定义了OXEC Immigration网站所有页面的Title和Meta Description优化策略。

---

## 元数据优化原则

### Title标签优化规则

- **长度**：50-60个字符（中文），确保在搜索结果中完整显示
- **关键词位置**：主关键词应在前30个字符内
- **品牌名称**：通常放在最后，用" - "或" | "分隔
- **格式**：主关键词 + 修饰词 + 品牌名称
- **避免**：关键词堆砌、过度优化、误导性标题

### Meta Description优化规则

- **长度**：150-160个字符（中文），确保在搜索结果中完整显示
- **内容**：简明扼要地描述页面内容，包含目标关键词
- **行动号召**：包含"咨询"、"了解"、"申请"等动词
- **独特性**：每个页面的描述应该独特，不重复
- **避免**：过度优化、关键词堆砌、与页面内容不符

## 实施步骤

### 步骤1：更新HTML中的Meta标签

在 `client/index.html` 中更新基础Meta标签：

```html
<meta name="description" content="OXEC移民事务所提供专业的加拿大移民咨询服务...">
<meta property="og:title" content="加拿大移民咨询 | 专业移民律师 | OXEC移民事务所">
<meta property="og:description" content="OXEC移民事务所提供专业的加拿大移民咨询服务...">
```

### 步骤2：更新页面组件中的Meta标签

在各页面组件中添加动态Meta标签更新：

```tsx
import { useEffect } from 'react';

export function SkillWorkerPage() {
  useEffect(() => {
    document.title = '加拿大技术移民 | Express Entry快速通道 | OXEC';
    document.querySelector('meta[name="description"]')?.setAttribute(
      'content',
      'OXEC提供专业的加拿大技术移民咨询。专业处理Express Entry、CRS评分...'
    );
  }, []);

  // 页面内容...
}
```

### 步骤3：验证Meta标签

使用以下工具验证Meta标签：

1. **Google Search Console** - 检查Google如何解析您的Meta标签
2. **Google Rich Results Test** - 验证结构化数据
3. **SEO检查工具** - 检查Meta标签长度和质量

实施元数据优化后，预期在3-6个月内看到以下改善：

- **点击率（CTR）提升**：20-30%（更吸引人的标题和描述）

- **排名提升**：部分关键词从20+位提升到10-15位
- **有机流量增长**：30-50%（更多用户点击搜索结果）

## 注意事项

1. **避免关键词堆砌** - 自然融入关键词，不要过度优化
2. **保持唯一性** - 每个页面的Title和Description应该独特
3. **准确描述内容** - Meta标签应准确反映页面内容
4. **定期更新** - 定期检查和更新Meta标签，保持相关性
5. **监测效果** - 使用Google Search Console监测排名和点击率变化
